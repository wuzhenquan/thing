

## 运行数据库

**在 macOS 上运行 mongoDB:** 

按 [run mongodb](https://docs.mongodb.com/manual/tutorial/install-mongodb-on-os-x/#run-mongodb-community-edition) 即可

命令行输入 `brew services start mongodb-community@4.2`，mongodb 就已经启动。

**操作 mongoDB 里的数据** 

如果需要操作数据库，命令行输入 `mongo` 

## 运行后端

```sh
yarn start
```

## 运行前端

```shell
yarn start
```
